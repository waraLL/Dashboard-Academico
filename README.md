Dashboard de Abandono y Rendimiento Estudiantil

Proyecto desarrollado en Power BI para analizar factores asociados al abandono estudiantil 
y rendimiento académico mediante visualizaciones interactivas y KPIs educativos.

Objetivos
- Identificar factores de riesgo relacionados con el abandono estudiantil
- Analizar variables como asistencia, estrés, retrasos y becas
- Facilitar la interpretación de datos educativos para la toma de decisiones
